# 🍔 LocalBite - UMKM Food Landing Page

LocalBite is a modern and interactive landing page designed to showcase trending local food and beverage businesses (UMKM) in a clean and elegant way.

## 🚀 Features

* Modern and responsive UI design
* Separate sections for food and beverages
* Smooth scrolling navigation
* Interactive menu navigation
* Clean card-based product layout
* Stylish footer with gallery

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 🎯 Purpose

This project was created as a front-end portfolio to demonstrate UI design skills and basic interactivity using vanilla JavaScript.

## 📸 Preview

Landing page showcasing modern UMKM food discovery platform.
